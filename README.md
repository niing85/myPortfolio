# My Portfolio

A modern personal portfolio website built with React, Vite, and Tailwind CSS. This project showcases a responsive landing page, project highlights, an about section, and contact links for Nuttawipha Tangroe.

## Features

- Responsive single-page layout with fixed navigation
- Smooth section scrolling for Home, Contributions, About, and Contact
- Downloadable resume link and contact copy-to-clipboard functionality
- Animated skill marquee banner and project cards
- Clean UI built with Tailwind CSS and Lucide icons

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React
- React Fast Marquee
- React Router DOM (dependency included for future route-based updates)

## Project Structure

- `src/App.jsx` — main application layout and section composition
- `src/Component/Navbar.jsx` — responsive navigation bar with mobile menu
- `src/Pages/Home.jsx` — introduction and skill showcase
- `src/Pages/Contributions.jsx` — project/contribution cards
- `src/Pages/About.jsx` — biography and career goals
- `src/Pages/Contact.jsx` — email, LinkedIn, and GitHub contact section
- `src/index.css` — Tailwind directives and scroll behavior styling

## Getting Started

### Prerequisites

- Node.js 18+ recommended
- npm or yarn installed

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

Open the local development URL shown in the terminal after Vite starts.


## Notes

- The navigation uses anchor links to scroll to different sections.
- Contact page includes a click-to-copy email button and external links to LinkedIn and GitHub.
- The project currently includes placeholder entries for additional future projects.
